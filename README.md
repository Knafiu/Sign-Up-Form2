This project is a responsive, two-column Sign Up Form webpage designed using HTML5 and CSS Flexbox.
It displays a full-height layout with an image on the left and the sign-up form on the right. The design emphasizes clean structure, readability, and modern layout techniques.

The goal of the project is to practice:

CSS Flexbox layout (row + column structures)

Responsive image handling using object-fit

Form styling, spacing, and alignment

Working with labels and input elements

Creating a clean, professional UI without frameworks
